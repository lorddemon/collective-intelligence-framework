# Data types supported by CIF #

By default CIF supports the following native data types:
  1. IP address
  1. FQDN
  1. URI
  1. CIDR
  1. Email address
  1. Malware hash (sha1/md5) (_must be enabled in feed config_)
  1. UUID (_generally used by the API_)
